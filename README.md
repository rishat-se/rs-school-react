# React. Components

## Downloading and installing

```
git clone https://github.com/rishat-se/rs-school-react.git
```

```
cd rs-school-react
```

```
git checkout ssr_ssg
```

```
npm install
```

## Running application

```
npm run start
```

After starting app you can open it on http://localhost:5173/

## Running test

```
npm run cy:e2e
```
